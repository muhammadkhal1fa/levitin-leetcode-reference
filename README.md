# Levitin's Algorithms → LeetCode

An interactive reference guide that maps the major algorithm design techniques from *Introduction to the Design & Analysis of Algorithms (3rd Edition)* by Anany Levitin to carefully selected LeetCode practice problems.

The goal of this project is to bridge theoretical algorithmic concepts with practical problem-solving by organizing LeetCode questions according to the techniques taught in the book.

## Features

- Clean modern UI with responsive design
- Organized by algorithmic paradigm and textbook chapters
- Difficulty-separated problem groups (Easy / Medium / Hard)
- Fast live search and filtering
- Direct links to LeetCode problems
- Sticky sidebar navigation
- Fully static and deployable via GitHub Pages

## Covered Techniques

- Brute Force & Exhaustive Search
- Graph Traversal (BFS / DFS)
- Decrease-and-Conquer
- Divide-and-Conquer
- Transform-and-Conquer
- Space-Time Trade-Offs
- Dynamic Programming
- Greedy Algorithms
- Iterative Improvement
- Backtracking

## Purpose

This project was built as a structured companion resource for students studying algorithm design and preparing for technical interviews or competitive programming.

Rather than practicing problems randomly, the platform encourages technique-oriented learning aligned with foundational algorithmic theory.
